Basic Information

  • Name: The pet's name.
  • Species: Indicate whether it’s a dog, cat, rabbit, etc.
  • Breed: Specify the breed if known (e.g., Labrador Retriever, Siamese Cat).
  • Age: Provide the pet's age in months.
  • Gender: Specify whether the pet is male or female.
  • Size/Weight: Include current size and weight, and the expected size if the pet is not fully grown.

Description - Optional but recommended details

Health Information

  • Vaccination Status: List vaccinations the pet has received (e.g., rabies, parvovirus).
  • Spaying/Neutering: Indicate whether the pet is spayed or neutered.
  • Microchip Status: Mention if the pet has been microchipped.
  • Medical History: Disclose any medical conditions, treatments, or ongoing needs (e.g., allergies, surgeries, special diets).
  • Deworming: Confirm if the pet has been dewormed.

Personality and Behavior

  • Temperament: Describe the pet’s behavior (e.g., friendly, playful, shy, energetic).
  • Training: Mention if the pet is trained (e.g., house-trained, leash-trained).
  • Compatibility: Indicate how the pet interacts with:
    • Children
    • Other pets (dogs, cats, etc.)
    • Strangers

Adoption Requirements

  • Living Environment: Specify the ideal home environment (e.g., house with a yard, apartment).
  • Activity Level: Mention if the pet requires high physical activity or is more sedentary.
  • Care Requirements: Highlight any special needs, such as grooming or dietary requirements.

Pet History

  • Rescue Story: Provide a brief background about the pet (e.g., rescued from a shelter or stray).
  • Previous Owners: If applicable, include any information about the pet’s prior living conditions.
